# Revo Coffee — учебный проект (Практическое задание №2)

Данный проект выполнен в рамках практического задания по теме **HTML5 и семантическая вёрстка**.
Цель работы — создать сайт по предоставленному макету, сформировать структуру
страницы из семантических блоков и наполнить её текстовым контентом.

## Функционал и структура проекта

Страница состоит из следующих основных разделов:

- **Header** — логотип, навигация, главный заголовок и вводный текст.
- **Основные секции (Main)**:
  - *Создайте свою кофейную базу* — текстовые блоки с описанием.
  - *Выберите свой любимый кофе* — карточки продуктов.
  - *Подарочные наборы* — демонстрационная текстовая секция.
  - *Комбо-наборы* — карточки товарных наборов.
  - *Контакты* — контактная информация.
- **Footer** — завершение страницы с копирайтом.

## Используемые технологии

В проекте использовались:

- **HTML5** — семантическая разметка  
- Работа со следующими инструментами:
  - Typograf (форматирование текста)
  - W3C Validator (валидация HTML-кода)
  - Git + GitHub (публикация и управление версиями)
